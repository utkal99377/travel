# Manali – Spiti – Kasol Travel Guide

A **premium mobile-first travel website** with 3D mountains, animated timeline, interactive route map, and smooth GSAP transitions.

## 🚀 Run Locally
1. Open the folder in VS Code.
2. Serve the folder (recommended):
   ```sh
   python -m http.server 8000
   ```
3. Open http://localhost:8000 in your browser.

## 📁 Contents
- `index.html` – Main page layout
- `style.css` – Custom glassmorphism + neumorphism styles
- `script.js` – Three.js, GSAP, ScrollTrigger logic
- `assets/images/` – Destination SVG artwork

## ✅ Features
- Mobile-first (320–480px optimized)
- Sticky header + bottom navigation
- 3D mountains, clouds, snowfall (Three.js)
- Scroll-based parallax & timeline animations (GSAP)
- Route map with animated dot and path
- Destination cards with 3D tilt hover
- Scroll progress indicator + back-to-top FAB
- Ambient audio toggle (Web Audio) and loading animation
